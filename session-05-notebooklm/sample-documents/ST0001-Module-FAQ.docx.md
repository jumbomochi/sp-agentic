## Module Information

**Q: What is ST0001 Statistics I about?**

ST0001 Statistics I is a foundational module that introduces students to statistical thinking and data analysis. Topics covered include descriptive statistics, probability, probability distributions, sampling distributions, hypothesis testing, confidence intervals, correlation, and simple linear regression. The module emphasises real-world applications and the use of statistical software for data analysis.

**Q: Who is the module coordinator?**

The module coordinator is Dr. Tan Wei Lin. You can reach her at tan_wei_lin@sp.edu.sg. Her office is located at Block T16, Room 05-12.

**Q: What are the module learning outcomes?**

Upon completion of this module, students should be able to:
1. Summarise and interpret data using descriptive statistics and graphical displays
2. Apply probability rules and probability distributions to solve real-world problems
3. Perform hypothesis tests and construct confidence intervals for population parameters
4. Analyse relationships between variables using correlation and simple linear regression
5. Use statistical software (Google Sheets / Excel) to conduct data analysis and interpret outputs
